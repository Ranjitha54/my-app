# React Shopping Cart

## Description
A ReactJS-based shopping cart app that fetches products from Fake Store API, allows users to add/remove items from the cart, adjust quantities, and view a discounted total.

## Tech Stack
- ReactJS
- React Router
- Tailwind CSS for styling

## Instructions to Run
1. Clone the repo.
2. Run `npm install` to install dependencies.
3. Run `npm start` to start the app on `http://localhost:3000`.
